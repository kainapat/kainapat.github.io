Functions

Functions are sub-programs within large programs, sometimes known as methods or routines, and are typically referred to as "functions" in structured programming, but a "method" in object oriented programming.

Types of functions are divided into two types:
1) Standard functions are available in the Python library.
2) Created by the programmer itself (User defined functions). For the functions that exist in the language when executing Python, the file must always be imported. This chapter discusses creating a function that the programmers created for their own use

Functions Names

Function naming rules:

Cannot use key words as a function name.
Cannot contain spaces.
First character must be a letter or underscore.
All other characters must be a letter, number or underscore.
Uppercase and lowercase characters are distinct

LOCAL VARIABLES

A local variable is created inside a function and cannot be accessed by statements that are outside the function. Different functions can have local variables with the same names because the functions cannot see each other's local variables.

Example (LOCAL VARIABLES)

def manin():
texas()
california()
def texas():
birds = 5000
print('texas has', birds, 'birds.')
def california():
birds = 8000
print('california has', birds, 'birds.')

PASSING ARGUMENTS

An argument is any piece of data that is passed into a function when the function is called. A parameter is a variable that receives an argument that is passed into a function.

Example (PASSING ARGUMENTS)

def main():
print('The sum of 12 and 14 is')
show_sum(12, 45)
def show_sum(num1, num2):
result = num1 + num2
print(result)
main()
