if-else

The if-else statement is used if there are two or more options. The run of the if-else statement starts by checking the condition and if the result is true it will execute the statement after the if statement, but if the result of the condition check is false. Run the command after else instead of if-else, the command format is as follows.

Example (if-else)

if condition:
statement(s)
else:
statement(s)

An if-elif-else process is characterized by a condition that overlaps the condition continuously depending on the complexity of the problem in the structure of the if-elif-else operation. Starting from the comparison of the first condition (1): if false will execute the statement after else (1): but if true the program executes a statement after page 123. The : after the : there are two conditions (1.1) that must be checked again if the result of the comparison is false. The program executes the statement after else (1.1): if the comparison result is true, it compares to the if statement in the 3rd condition (1.1.1), but if the comparison results in the 3rd time it is false. The fourth comparison is performed with the condition (1.1.2): if the fourth comparison is false, the statement after else (1.1.1) is executed; if true, it is executed. Execute the command after condition (1.1.2): etc.
