Repetition Structure

An iterative structure is one in which specific statements or groups of instructions are repeated more than once. This iteration must always be determined as a condition to determine whether or not to loop.

While Loop

An iterative statement with a precondition that always checks the conditions before execution. When the checked condition is true, the statement after "while" is executed, but if the condition is false it will stop the order.

Calculating Running Total

The total is the sum of the numbers from each iteration. The variable that stores the sum is known as the accumulator.

Two of these components are often used in programs that calculate the sum of a set of numbers:

- a loop that iterates over each number in a series

- A variable that collects all numbers when reading

Sentinel

A special value that marks the end of a value series.

- At the end of each loop, ask the user. If there are other values to be taken, however, asking this question at the end of each loop might make the program cumbersome for the user.

- Asks the user at the beginning of the program how many items are in the sequence. This can be cumbersome for the user if the sequence is very long and the user can't remember how many items are in it so, the user will need to count the number of items.

Input Validation Loops

Input validation is the process of ensuring that information entered into a program is correct before it is used in calculation. Input validation is often done through the use of loops that run as long as the input variable refers invalid data.

Example (Input Validation Loops)

score = int(input("Enter a test score: "))
while score < 0 or score > 100:
print("ERROR: The score cannot be negative")
print("or greater that 100.")
score = int(input("Enter the correct score: "))

Nested loops

Python allows programmers to use nested loops (placement of a “for loop” within a “for loop”) as it shows below.

Nested loops of "for"

for iterating_var insequence:
for iterating_var insequence:
statements(s)
statements(s)

Nested loops of "while"

while expression:
while expression:
statement(s)
statements(s)

Nested loops of "for" and "while"

for iterating_var insequence:
while expression:
statement(s)
statements(s)

The number of levels that can be nested loops in Python is not limited. As a result, programmers may nest loops independently. It depends on the complexity of the task, but overlapping loops might make it difficult for the program to understand or fix errors. There will be an indentation issue within the loop. This may cause the program to malfunction therefore, this is a weakness for Python since the lack of a boundary marker often requires depend on the eye to determine the limit of a command.

Example (Nested loops)

for i in range(1, 10):
print("For loop for i")
for j in range(1, i):
print("For loop for j")

Continue , Break , Pass

Continue

A statement instructing the program to restart at the beginning of the loop, resulting in any statements after the continue statement and within the scope of the repeat statement not being processed in that loop. In the continuing scheme, the continue statement works with both "while" and "for loop".

Example (Continue)

for letter in "Kainapat Suwannachote" :
if letter == "a" or letter == "k" :
continue
print("Current Letter :", letter)

Resultant (Continue)

Break

This command tells the program to stop repeating itself. When the program stops in the current loop during the execution of the break statement in a nested loop, the program continues to run in the loop that encapsulates the current loop. The break statement only works with the while and for loops, not if, if-else, and if-elif as in C for a break mockup.

Example (Break)

for letter in "Kainapat Suwannachote" :
if letter == "a" or letter == "k" :
break
print("Current Letter :", letter)

Resultant (Break)

Pass

It is one instruction that indicates that no action on that line is taken. Often used when a program is organized but the instructions have not been specified in detail.

Example (Pass)

for letter in "Kainapat Suwannachote" :
if letter == "a" or letter == "k" :
pass
print("Current Letter :", letter)

Resultant (Pass)
